# EconSim - Quick PRD

Build a locally-run, single-page web app for MBA students to interactively explore macroeconomic models. Users adjust individual variables and immediately see how changes propagate across three simultaneously displayed graphs: the IS/LM model, the Money Market, and the AD/AS model. Formulae for each curve are displayed using symbolic variables with subscript notation (e.g., Y₀, r₀); no real numbers are used. Variable controls are simple up/down/same toggles with fixed step sizes.

Tech stack: HTML + CSS + vanilla JavaScript + Plotly.js (via CDN or local bundle). No backend, no build step, no installation required — users clone from GitHub and open `index.html` in any modern browser.

Out of scope: user accounts, session persistence, mobile support, data export, additional macro models (Phillips Curve, Mundell-Fleming, etc.), and any real numerical simulation.
